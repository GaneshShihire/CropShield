# **App Name**: AgriVision

## Core Features:

- Crop Price Prediction: Predict future crop prices based on historical data, regional trends, and weather forecasts, utilizing an AI model. The model incorporates government scheme data as a tool to provide a holistic view.
- Leaf Disease Detection: Identify plant diseases from uploaded leaf images using Convolutional Neural Networks (CNNs), and provide recommended treatments.
- Regional Data Input: Allow Talathi (local officials) to input regional agriculture data to improve prediction accuracy.
- Weather Forecast Integration: Display current and predicted weather conditions for specific regions, impacting crop price predictions and disease risk assessments.
- Government Scheme Information: Provide information and updates on relevant government schemes and initiatives for farmers.
- User-Friendly Interface: Design an intuitive and easy-to-use interface for farmers and officials to access all features.
- Continuous Model Refinement: Regularly update and refine AI models based on new data and user feedback to ensure accuracy and reliability.

## Style Guidelines:

- Primary color: Earthy green (#74A860) to evoke a sense of nature and growth.
- Background color: Light beige (#F5F5DC) to provide a clean, neutral backdrop.
- Accent color: Warm orange (#FFA500) for important calls to action and highlights.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a blend of modern and approachable aesthetics.
- Use clear and representative icons for crops, weather conditions, and diseases. The iconography should follow a consistent outline style.
- Implement a clean and organized layout with clear sections for each feature (price prediction, disease detection, weather forecast, schemes). Use cards to present individual pieces of information.
- Subtle animations to enhance user experience, such as smooth transitions and loading indicators. Avoid distracting animations.
